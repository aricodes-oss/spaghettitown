# Spaghettitown

The official website for [Choosing Clarity](https://spaghettitown.xyz), a rock band based in Portland, OR.

Live site: [spaghettitown.xyz](https://spaghettitown.xyz)

## Tech Stack

- [Next.js](https://nextjs.org) — React framework
- [Mantine](https://mantine.dev) — UI component library
- [OpenNext for Cloudflare](https://opennext.js.org/cloudflare) — deployment adapter
- [Cloudflare Pages](https://pages.cloudflare.com) — hosting

## Development

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Preview on Cloudflare Runtime

To preview the app running on the Cloudflare runtime locally:

```bash
npm run preview
```

## Deploy

```bash
npm run deploy
```

Deploys to Cloudflare Pages via OpenNext.
